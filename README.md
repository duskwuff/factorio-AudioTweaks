Audio Tweaks
============

Allows the player to simplify or disable certain categories of in-game sounds:

* **Item pickup sounds** (including when selecting an item from the quickbar). Setting this to "basic sounds" uses a single sound when selecting any item, rather than the unique per-item sounds which were introduced in Factorio 2.0. Setting it to "no sounds" does exactly what you'd expect.

* **Item transfer/drop sounds** (including when quick-tranferring items to or from a chest, or when deselecting an item in the cursor). As with pickup sounds, "basic sounds" uses a single sound for all items, and "no sounds" disables these sounds.

* **Tool selection/deselection sounds** (e.g. when selecting entities to copy). You probably want to leave this on "default".

* **Entity open/close sounds** (e.g. when opening an assembler, chest, or car). Setting this to "basic sounds" will use a much smaller set of sounds than default (e.g. one sound for any type of machine), and "no sounds" will make opening/closing entities silent.

* **Night vision sounds** can be disabled if you don't like hearing the little electronic whine at sunrise/sunset.

***By default, this mod does almost nothing.*** To enable any of the changes described above, open the mod settings (under "Startup").
